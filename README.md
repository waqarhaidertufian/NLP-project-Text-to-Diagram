# Text to Diagram Application

A modern, user-friendly web application that converts text to diagrams using Mermaid.js syntax. Designed to run in Visual Studio Code with excellent UX.

## Features

- **Real-time Diagram Generation**: Convert text to diagrams instantly
- **Multiple Diagram Types**: Flowcharts, Sequence Diagrams, Class Diagrams, Pie Charts, Gantt Charts, and more
- **Excellent UX**: Clean, modern interface with dark/light mode
- **Export Options**: Save diagrams as PNG, SVG, or PDF
- **VS Code Integration**: Designed to run seamlessly in VS Code
- **Responsive Design**: Works on desktop and tablet devices
- **Syntax Help**: Built-in guidance for Mermaid.js syntax
- **Keyboard Shortcuts**: Quick actions for power users

## How to Run in Visual Studio Code

### Method 1: Using Live Server Extension (Recommended)

1. **Install Live Server Extension** in VS Code
   - Open VS Code
   - Go to Extensions (Ctrl+Shift+X)
   - Search for "Live Server" by Ritwick Dey
   - Install it

2. **Open the Project Folder**
   - Open VS Code
   - Select "File" â†’ "Open Folder"
   - Navigate to and select the `text-to-diagram-app` folder

3. **Run with Live Server**
   - Right-click on `index.html` in the Explorer
   - Select "Open with Live Server"
   - The application will open in your default browser

### Method 2: Direct Browser Opening

1. **Open the Project in VS Code**
   - Open the `text-to-diagram-app` folder in VS Code

2. **Open in Browser**
   - Right-click on `index.html`
   - Select "Reveal in File Explorer" (or "Reveal in Finder" on Mac)
   - Double-click `index.html` to open in your browser

### Method 3: Using VS Code's Built-in Preview

1. **Install "Open in Browser" Extension** (optional)
   - Search for "open in browser" in VS Code extensions
   - Install the extension

2. **Preview the HTML File**
   - Open `index.html` in VS Code
   - Press `Alt+B` (or use the right-click context menu) to open in browser
